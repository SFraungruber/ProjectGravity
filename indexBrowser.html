About ProjectGravity

Short Info

The aim of this project is to control a game in the browser of a PC via a smartphone. We develop an Android, iOS application and an additionally web backend for communication and a web frontend to display the game. Communication takes place via the Internet and by client-side input of a pin, which is displayed in the web frontend, to established the connection.

Creation

The project was created in cooperation with the University Of Applied Sciences Upper Austria Hagenberg.

What you see on this webpage and the both apps for Android and iOS were createt and published within the summer term in 2015.

If you like our project, have sugestions or other informations, write us at ProjectGravity1@gmail.com
